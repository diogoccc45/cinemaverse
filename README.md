# 🎬 cinemaverse — letterboxd data science project

Análise do meu perfil Letterboxd combinando ciência de dados, ciência de redes e machine learning.

## roadmap

- [x] Passo 1 — explorar o dataset raw
- [ ] Passo 2 — activar e testar a API do TMDb
- [ ] Passo 3 — enriquecer os 768 filmes com metadados
- [ ] Passo 4 — análise exploratória com os novos dados
- [ ] Passo 5 — ciência de redes (grafo de actores)
- [ ] Passo 6 — machine learning (embeddings + clusters)
- [ ] Passo 7 — dashboard final interactiva

## dados

- **768 filmes** vistos (Jul 2023 – Mai 2026)
- **767 ratings** (média: 3.32 / 5.0)
- **417 reviews** escritas em PT/EN
- **418 entradas** no diário
- **157 likes**, **31 favoritos**, **248 na watchlist**
- **12 listas** curadas (incluindo 157 filmes A24)
